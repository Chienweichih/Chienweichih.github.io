横板正手攻球要点  
1. 右前脚掌向右蹬转70度，重心压在右腿，顺势把前臂展开  
2. 拍头引领手指手腕前臂向前上方挥拍击球  
3. 重心从右腿蹬转至左腿，手法与步法同时还原至准备姿势  
4. 击球点在身边的斜前方，球，手，身体成为一个三角形  
5. 一定要迎前击球，不能等，在身边的前面接触球

### EP09 击球技术 直板推挡

乒乓球教学 击球技术 直板推挡要点：

4. 击球点在身体前面，击球中上部  
5. 迎前击球的部位在起跳点和最高点

### EP17 乒乓球步法要点 单跨步

乒乓球教学 步法 单跨步注意事项  
1. 单跨步适用于比较急，比较快，角度又比较大的来球  
2. 向右单跨时，左腿要先垫一小步，右腿向高大跨出，跨出脚落地后马上就蹬回来，反之亦然  
3. 单跨步落地之时，也是击球的时候，手脚一定要同步打出去，随即还原，要做到一气呵成  
4. 从左向右单跨步时，左腿垫步大一点，右腿就可以照顾到更大面积，从右向左单跨步也是同样道理  
5. 步法移动时要前脚掌着地  
6. 每打完一板，手上动作也要还原

### EP19 乒乓球步法 小碎步动作示范详解

乒乓球教学 步法 小碎步注意事项  
1. 小碎步是步法中的润滑剂，使用小碎步可以抢到最佳击球点，抢到最好的位置  
2. 在小碎步的使用中，首先要树立一个意识，就是每一个球我都要抢到最佳位置  
3. 小碎步的练习中，最重要的就是练习速度。  
4. 小碎步的使用当中，包括击球前的准备以及击球后的还原，身体一直是在颠着，这样才能保证抢到最佳的击球点。

### EP20 站位步法 削球步法

乒乓球教学削球步法注意事项：

4. 并步并非只有左右移动，还可以前后移动，从而运用在削球打法中

### EP21 身法 正手位半台走位

乒乓球教学 身法讲解  
1. 第一阶段用手打球，第二阶段用身体打球，第三阶段用心打球  
2. 用身体打球的意识指导下，可以对命中率以及回球质量，有一个很好的保障  
3. 用身体打球需要全身协调配合，右前脚掌发力，腰部协调控制，上身重心保持稳定  

正手位半台走位注意事项  
1. 练习时首先眼睛要盯住对手的拍型，判断来球落点，然后移动步法，在身前击球  
2. 要做到眼，手，脚协调配合，关键之处在与右胯，右胯可以很好的衔接手法与步法  
3. 目标时手上可以有轻有发力，轻轻重重连续击球步法轻盈移动的同时还要脚跟下生根  
4. 眼睛一定要盯住对手的拍型，知道自己要往哪个方向走位，右前脚掌蹬转要充分，胯作为轴心一定要保持上身的稳定  
5. 每一板结束后，手法，步法都要及时还原，这是下一板的保障和基础

### EP29 控球技术 控制球左右

乒乓球教学 控制球左右注意事项：  
1. 正手打斜线时，首先要用身体包裹住球其次再用球板，板型包裹住球，击球的右侧。  
2. 在正手旋球的对攻中，击球中上部的右侧，可以打出斜线球。  
3. 正手打直线时，首先用小碎步调整站位，身体球板都对着直线位置，然后击球的中上部。  
4. 正手打斜线时，身体可以离球近一些，这样右腿的蹬转也能往斜线发力  
5. 正手打直线时，动作发力方向要对着直线，板型在触球时由于线短，完成速度要比斜线快。  
6. 反手位包裹到球的左侧就可以打出斜线，手找球的位置要精准，根据打过去球落点来调整拍型

### EP32 发球技巧练习 乒乓球发球要点 发球战术

教你打乒乓球
本章节重点介绍乒乓球发球技巧，在乒乓球教学中占很重要一部分。  
乒乓球发球的技巧：  
1. 练习发短球时，要尽量让球落在对方半台后，最少跳三跳。  
2. 练习发球时，最好长短结合一起练。  
3. 偷长球时击球点不能过高，重心可以降低一些，等球落得低一点时再触球。  
4. 发球时要控制好节奏，可以慢一点，想好了再发。  

乒乓球发球注意事项：  
1. 发球配套体现在三个方面：落点上的配套，旋转上的配套以及节奏上的配套  
2. 发球时首先上身重心要下来，随后沉肩，坠肘，手心在台面上，垂直向上抛弃16公分，球落到离台面上。垂直向上抛球16公分，球落到离台面较近时再击球。  
3. 发长球时第一跳要落在本方球台端线位置，发短球时，第一跳要尽量靠近网根。  
4. 发加转球要摩擦球的底部，发不转球要摩擦球的上部。  
5. 发侧旋球时要注意握拍方式，一定要把板把立起来，这样才能展开手腕，才能发上力。接触球时摩擦球的左面。  
6. 发球时可以进行节奏的变化，先发慢的短的，然后再偷长的快的，从而给对手制造麻烦。

### EP33 发球抢攻技术要点

乒乓球发球抢攻注意事项  
1. 发球抢攻要根据自己的技术特长来设计  
2. 发球技术好，抢攻难度小，发球质量不高，抢攻难度就高  
3. 发球抢攻分两种，如果偷长球就面临上旋球抢攻，如果以发短为主，那么抢攻下旋球就会比较多  
4. 实战中发球抢攻上旋与抢攻下旋配套使用  
5. 发完球之后重心不要起来，始终保持进攻的状态，一旦来球之后就可以直接进攻  
6. 发球抢攻的设计当中，要尽量搜索规律，保障自己发出的球，对方再回击大批本方半台时，可选择的范围越少越好

### EP34 接发球练习注意事项 接球技术技巧

乒乓球接发球技术要点  
1. 接发球以稳为主，稳中带凶  
2. 对方球拍触球的一刹那，这是接发球判断的第一信息，球在运行过程中弧线的高低时第二信息，球落台后的弧线变化时第三信息。  
3. 接发球首要任务是判断，实战中尽量把握第一信息，实在不行再往后延。  

乒乓球接发球注意事项：  
1. 接发球上策：积极主动，先上手为主  
2. 接发球中策：控制动手，用摆短制约对手打发球抢攻，不让对手抢先上手。  
3. 接发球下策：主动劈长，对手轻拉之后，我们再来衔接第二板的反冲，主动去防守。  
4. 在接短球的时候，手不能低，否则球容易冒高。重心要插到台子里。  
5. 接发球时要根据来球灵活调整步法，对方发短球可以通过并步上台内接。对方发场球，可以用小碎步往后退，找到最佳位置从容出售。  
接正手位短球时，上半身的重心要保持稳定下半身的重心要提起来，脚后跟抬起，这样拉球可以抢到高点，搓球时也有空间可以控制的住球。
